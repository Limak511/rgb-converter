# RGB Converter

A simple C# console application that converts RGB color values between the 0–1 float range and the 0–255 integer range. Perfect for quick color conversions when working with different color representations.



## Features

### Bi-directional Conversion:

Convert normalized floats (0.0 to 1.0) to 8-bit integers (0 to 255).

Convert 8-bit integers to normalized floats (rounded to 3 decimal places).
        


## Requirements

.NET 9.0



## How to Use

### Clone the repository:
    git clone https://github.com/Limak511/rgb-converter.git

### Navigate to the folder:
    cd rgb-converter

### Run the application:
    dotnet run

### Follow the on-screen prompts:

Select your conversion type (1, 2, or 0 to exit).

Enter your Red, Green, and Blue values when prompted.

The result will be displayed clearly before returning to the main menu.
